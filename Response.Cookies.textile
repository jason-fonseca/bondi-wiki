h2. Objects

<p>
As cookie values are assigned they are replaced by an internal cookie object and the cookie value is placed onto the internal cookie object value. This object these supported properties:
* Value
* Expires
* Domain
* Path
* Secure
* HttpOnly
* Comment
* Version
</p>

h2. Methods

<p>
The [[Response.Cookies]] object also supports a special method, Encrypt. When used, this function will assign a property to the [[Response.Cookies]] object that has its value encrypted. (See also [[Request.Cookies]].Decrypt)
</p>
<p>
,<b>Usage.<b/> Response.Cookies.Encrypt('<property name>', <Key Object>, <Plaintext>)
</p>
