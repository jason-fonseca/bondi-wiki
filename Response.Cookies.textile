h2. Objects

<p>
As cookie values are assigned they are replaced by an internal cookie object and the cookie value is placed onto the internal cookie object value. This object these supported properties:
* Value
* Expires
* Domain
* Path
* Secure
* HttpOnly
* Comment
* Version
</p>

h2. Methods

<p>
The [[Response.Cookies]] object also supports a special method, Encrypt. When used, this function will assign a property to the [[Response.Cookies]] object that has its value encrypted. (See also [[Request.Cookies]].Decrypt)
</p>
<p>
,<b>Usage.<b/> Response.Cookies.Encrypt('<property name>', <Key Object>, <Plaintext>)
</p>

h2. Examples

@Response.Cookies.Test = "Hello World";@
@Response.Cookies.Test.Expires = (new Date(Date.parse(Date()) + 5000)).toGMTstring();@
@Response.Cookies.Test.Secure = true;@
@Response.Cookies.Test.HttpOnly = true;@

@Response.Cookies.Encrypt(Test2, {Key: 'Key', Algorithm: 'BLOWFISH', Hash: 'SHA1'}, 'Hello World');@
@Response.Cookies.Encrypt(Test3, {Key: 'Key2', Algorithm: 'AES', Hash: 'MD5'}, Test);@

// To see the values
@Response.Write("Response.Cookies.Test =" + Response.Cookies.Test);@
@Response.Write("Response.Cookies.Test2 = " + Response.Cookies.Test2);@
@Response.Write("Response.Cookies.Test3 = " + Response.Cookies.Test3);@
