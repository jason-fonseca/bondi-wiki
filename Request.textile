h2. Request

Instrinsic Request object available in bondi

The Request object also operates as a stream of bytes the represent any posted data. e.g. Response.BinaryWrite(Request) will reflect any input to the output byte for byte.

h2. Objects

* Request.QueryString
* Request.Cookies
* Request.Form
* Request.ServerVariables

h2. Properties

* Request.TotalBytes
* Request.Charset

h2. Methods

* Request.BinaryRead

h2. Examples

@code@
for(var x in Request.ServerVariables)
{
	Response.Write(x + ": " + Request.ServerVariables[x]);
}
