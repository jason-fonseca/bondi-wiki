h2(#summary). Summary

Constructable object available in [[Bondi|Bondi Reference]]

h2(#syntax). Syntax

@var html = new HTMLDocument([stream or string]);@

h2(#parameters). Parameters

- stream := A Bondi stream object (eg. from File, HttpClient, FtpClient etc) containing html data.
- string := A string containing html text.

h2(#description). Description

The HTMLDocument object is aimed to be an implementation of the DOM (Document Object Model) commonly provided by most web browsers.
