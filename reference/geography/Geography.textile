h2(#methods). Methods

- CountryFromIP(ip address) := Returns null if resolution of ip wasn't possible or the "country code":http://www.maxmind.com/app/iso3166 for given ip address.
- RangeFromIP(ip address) := Returns null if resolution of ip wasn't possible or a range object with properties: Start, End.
- ResolveFromIP(ip address) := Returns null if resolution of ip wasn't possible or a geography object with properties: CountryCode, CountryCode3, Region, City, ContinentCode, PostalCode, Longitude, Latitude.
