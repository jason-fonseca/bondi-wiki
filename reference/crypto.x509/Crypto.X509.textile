h2(#summary). Summary

Property with attached methods on the [[Crypto]] object.

h2(#description). Description

h2(#methods). Methods

- Create() := Creates and returns a randomly generated X509 certificate.
- Open(stream or filename) := If passed an object will attempt to read PEM data from the stream, if given a string will instead try to open an X509 certificate from the filesystem.
- Sign() := Not Implemented.

h2(#examples). Examples

```javascript
// Creating X509 certificate
var x509 = Crypto.X509.Create();
Response.Write("Created new X509 Cert:\n\
\tSubject: " + x509.Subject + "\n\
\tIssuer: " + x509.Issuer + "\n\
\tVersion: " + x509.Version + "\n\
\tSerial: " + x509.Serial + "\n\
\tSignature: " + x509.Signature + "\n\
\tNotBefore: " + x509.NotBefore + "\n\
\tNotAfter: " + x509.NotAfter + "\n");

var signature = Crypto.Sign(x509, data);
Response.Write("X509 Signature: " + Base64.Encode(signature) + "\n");

// X509 Verification
var valid = Crypto.Verify(x509, data, signature);
Response.Write("X509 Verified: " + valid + "\n");

// Opening X509 certificate
var x509FromFile = Crypto.Open("somecert.pem");
```
