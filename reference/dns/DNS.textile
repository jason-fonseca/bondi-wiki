h2(#summary). Summary

Instrinsic object available in [[Bondi|Bondi Reference]]

h2(#description). Description

h2(#properties). Properties

- BYPASSCACHE (read-only) := Always set to true at this stage.

h2(#methods). Methods

- A(hostname [, flags]) := Takes a hostname and optionally dns flags, returns null if hostname can't be resolved or an ip address (as a string).
- MX(hostname [, flags]) := Takes a hostname and optionally dns flags, returns null if hostname can't be resolved or an array of MX records that have properties Exchanger and Preference.
- NS(hostname [, flags]) := Takes a hostname, returns null if hostname can't be resolved or returns an array of string NS records.
- TXT(hostname [, flags]) := Takes a hostname, returns null if hostname can't be resolved or an array of string TXT records.

h2(#examples). Examples

```javascript
var ipAddress = DNS.A("www.google.com");
Response.Write("IP: " + ipAddress);

var mxRecords = DNS.MX("www.google.com");
Response.Write("MX: " + mxRecords.toSource() + "\n");

var nsRecords = DNS.NS("www.google.com");
Response.Write("NS: " + nsRecords.toSource() + "\n");

var txtRecords = DNS.TXT("www.google.com");
Response.Write("TXT: " + txtRecords.toSource() + "\n");
```
