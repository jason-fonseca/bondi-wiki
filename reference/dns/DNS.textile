h2(#methods). Methods

- A := Takes a name - returns an ip address (as a string)
- MX := Takes a name - returns an array of objects that have exchanger and preference properties
- TXT := Takes a name - returns the txt data against the record in the dns
- NS := Takes a name - returns the NS record
- SOA := Takes a name - returns an SOA record
